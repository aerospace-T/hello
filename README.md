一、计算机与编程语言

学这门课可以知道计算机能做什么、不能做什么、擅长做什么、不擅长做什么。
计算机如何解决问题的？需要告诉计算机一步一步怎么做，对人说：What to do. 对计算机说：How to do.


计算机语言

程序：用特殊的编程语言写出来表达如何解决问题的，不是用编程语言来和计算机交谈，而是描述要求它如何做事情的过程或方法。计算机只懂数字，数字 → 程序 → 算法，逐渐往顶层走。

算法：我们要让计算机做计算就需要找出计算的步骤，然后用编程语言写出来，计算机做的所有的事情都叫作计算，计算的步骤就是算法。

程序就是数据加计算。计算机做的所有的事情都叫做计算，计算的步骤就是算法；算法当中有一部分是套路，这需要不断地去阅读去交流去实践才能积累。


计算机的思维方式
枚举算法、二分法

程序的执行有两种方案：①解释:借助一个程序，那个程序能试图理解你的程序然后按照你的要求执行。②编译:借助一个程序，就像一个翻译，把你的程序翻译成计算机真正能懂的语言 ——机器语言—— 写的程序，然后，这个机器语言写的程序就能直接执行了

计算机语言本身是没有解释和编译之分，只是常用的执行方式不同而已；解释型语言有特殊的计算能力；编译型语言有确定的运算性能


about C

C 语言，即 C programing language，是从 B 语言发展而来，B 语言是从 BCPL 发展而来的，BCPL 是从 FORTRAN (人类历史上的第一个高级编程语言) 发展而来的。BCPL 和 B 都支持指针间接方式，所以 C 也支持了。C 语言还受到了 PL/I 的影响，还和 PDP-II 的机器语言有很大的关系。1973 年 11 月，第四版的 Unix(System Four)发布了，这个版本是完全用 C 语言重新写的。

1989 年 ANSI 发布了一个标准—ANSI C，1990 年 ISO 接受了 ANSI 的标准——C89。C 的标准在 1995 年和 1999 年两次更新——C95 和 C99

C 语言用在哪里? ①操作系统 ②嵌入式系统 ③驱动程序 底层驱动 图形引擎、图像处理、声音效果等



二、编程软件

C 需要被编译才能运行，所以你需要编辑器和编译器，或者，IDE（集成开发环境，integrated development environment）。集成开发环境 IDE = 编辑器 editor + 编译器 complier

推荐的编程软件：Dev C++，它免费、安装简单、不用建工程，但使用时需注意得将文件保存成 cpp 格式才能进行调试
